## 一、Onboarding 注册引导（4页）

| # | 页面 | 核心内容 |
|---|------|--------|
| O1 | Welcome | 品牌介绍 + "Get PSLE Ready" + 开始按钮 |
| O2 | Select Level | 选择年级 P1-P6，选择目标（PSLE备考/日常提升） |
| O3 | Select Subjects | 勾选科目 Math/Science/English/Chinese，可多选 |
| O4 | Create Profile | 输入孩子名字、头像选择、家长邮箱（可选） |

## 二、Home 首页（1页）

| # | 页面 | 核心内容 |
|---|------|--------|
| H1 | Home Dashboard | 今日学习概览：已完成题数/正确率/学习时长/连续打卡天数<br>每日一练入口（今日Grammar/Vocab/Oral任务）<br>快速操作：Scan Paper / Ask AI / Error Book<br>待复习错题提醒（最多显示3题）<br>本周进步趋势小图表 |

---

## 三、AI Tutor AI老师（5页）

| # | 页面 | 核心内容 |
|---|------|--------|
| T1 | AI Tutor Entry | 选择科目 → 拍照/相册/打字输入 → 最近对话历史 |
| T2 | Camera | 暗色取景框 + 拍照提示 + 快门/相册/闪光灯 |
| T3 | Photo Confirm | 拍照预览 → Process & Ask AI / Save to Error Book / Add more pages |
| T4 | AI Chat | 苏格拉底对话界面<br>左栏(iPad)：题目原图+OCR文字+科目标签<br>右栏(iPad)：AI对话+提示按钮(Hint/Got it)+完成保存<br>手机：全屏对话 |
| T5 | Chat Summary | 对话完成后总结卡：本题知识点、方法要点、掌握度评分、保存到错题本确认、推荐相似题 |

---

## 四、Error Book 错题本（4页）

| # | 页面 | 核心内容 |
|---|------|--------|
| E1 | Error Book Overview | 按学科分类卡片（Math/Science/English/Chinese）+ 总数统计<br>薄弱知识点Top 3摘要<br>全局筛选：按日期/掌握状态/科目 |
| E2 | Error List | 某科目下的错题列表<br>按知识模块筛选（Ratio/Fraction/Speed...）<br>每题显示：缩略图+题目摘要+错因标签+掌握状态<br>多选checkbox → 批量导出PDF / 批量删除 |
| E3 | Error Detail | 题目标签（科目/模块/题型/来源/难度）<br>原图 + 清洁后图（可切换）<br>OCR提取的题目文字<br>AI错因分析卡（高亮显示）<br>操作：Ask AI Tutor / Mark Mastered / Delete<br>相似题推荐（Phase 2） |
| E4 | Export PDF | 选择的题目列表预览<br>导出设置：是否含原图/是否含AI分析/纸张大小<br>导出 → Share sheet（Files/WhatsApp/WeChat/Print） |

## 五、Learn 学习体系 — Math 数学（4页）

| # | 页面 | 核心内容 |
|---|------|--------|
| LM1 | Math Home | 知识模块网格（按P3-P6分级展示）：<br>Whole Numbers / Fractions / Decimals / Percentage / Ratio / Speed / Algebra / Geometry / Data Analysis / Problem Solving<br>每个模块显示：完成进度/掌握度百分比 |
| LM2 | Module Detail | 模块内三个Tab：<br>**📖 Knowledge Guide** 知识总结（概念/公式/规则）<br>**📝 Question Types** 题型指导（PSLE常考类型列表）<br>**🔧 Methods** 方法指导（每种题型的解题步骤） |
| LM3 | Question Type Detail | 某个具体题型的详情：<br>题型说明 + 解题方法（图文步骤）<br>例题示范（带AI讲解）<br>真题练习列表（来源标注学校/年份） |
| LM4 | Practice Question | 做题界面：<br>题目展示（文字+图片）<br>答题区（输入答案 / 选择题选项）<br>提交 → 对错判断 → AI讲解（苏格拉底引导）<br>做错自动加入错题本 |

## 七、Learn 学习体系 — English 英文（5页）

| # | 页面 | 核心内容 |
|---|------|--------|
| LE1 | English Home | 三大技能模块：<br>📗 Grammar（语法）<br>📘 Vocabulary（词汇）<br>🎤 Oral（口试）<br>每日一练进度条 + 连续打卡天数<br>PSLE题型专项训练入口 |
| LE2 | Grammar Module | 语法知识点列表（按P1-P6递进）：<br>Tenses / Subject-Verb Agreement / Prepositions / Conjunctions / Sentence Structures...<br>每个知识点：讲解 + 练习 |
| LE3 | Vocabulary Module | 词汇积累：<br>按主题分类（School / Family / Nature / Science...）<br>每日新词卡 + 例句 + 用法<br>词汇测试 |
| LE4 | Oral Practice | 口试练习：<br>Reading Aloud 朗读练习（显示短文 + 录音 + AI评分）<br>Stimulus-based Conversation 看图说话（显示图片 + AI对话引导） |
| LE5 | PSLE English Training | PSLE题型专项：<br>Comprehension (MCQ + Open-ended)<br>Grammar MCQ / Grammar Cloze<br>Vocabulary MCQ / Vocabulary Cloze<br>Synthesis & Transformation<br>Situational Writing<br>Continuous Writing<br>每个题型：方法指导 + 真题练习 |

## 九、Daily 每日一练（3页）

| # | 页面 | 核心内容 |
|---|------|--------|
| D1 | Daily Overview | 今日任务卡片（按科目）：<br>🟢 English Grammar — 5 questions<br>🟢 English Vocab — 3 new words<br>🟢 Chinese 词汇 — 3 new words<br>🟡 Chinese 语法 — 5 questions<br>已完成/未完成状态<br>连续打卡天数 + 本周完成率 |
