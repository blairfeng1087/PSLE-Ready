## 二、API 成本明细

### 1. Gemini 2.5 Flash（AI 核心）

**用途：** OCR 识别题目 + Tutor 辅导

| 场景 | 调用方 | 输入 | 输出 | 单次成本估算 |
|------|--------|------|------|------------|
| OCR 识别（管理员上传试卷） | 管理员 | ~1MB 图片 + 500 token prompt | ~500 token | ~$0.002 |
| Tutor 对话（学生提问） | 学生 | ~200 token（含历史） | ~150 token | ~$0.0003 |
| Tutor 对话（含图片） | 学生 | ~1MB 图片 + 200 token | ~150 token | ~$0.002 |

**Gemini 2.5 Flash 定价（2026）：**
- 输入：$0.15 / 百万 token（文本），$0.10 / 百万 token（图片）
- 输出：$0.60 / 百万 token

**月度估算：**

| 场景 | 频率 | 月调用量 | 月成本 |
|------|------|---------|--------|
| 管理员 OCR 录题 | 每天录 10 道 | ~300 次 | ~$0.60 |
| 学生 Tutor（纯文字） | 每人每天 5 轮对话 | ~7,500 次 | ~$2.25 |
| 学生 Tutor（带图片） | 每人每天 2 次 | ~3,000 次 | ~$6.00 |
| **Gemini 小计** | | ~10,800 次 | **~$8.85/月** |

**Gemini 免费额度：** 每分钟 15 次请求，每天无限（绑卡后）。50 个用户不太可能同时请求，免费额度基本够用。

## 六、省钱建议

1. **Gemini 免费额度用满** — 绑卡后每天免费额度很高，50 用户基本够
2. **Tutor 限流** — 每人每天限制 Tutor 对话次数（如 20 次）
3. **缓存常见问题** — 同样的题目第二次问，返回缓存的回答
4. **有道按需使用** — Auto Clean 不自动触发，用户手动点击才调用
5. **图片压缩** — 上传前压缩图片，减少 Gemini 图片 token 消耗

---

*本文档为估算，实际成本取决于用户使用频率。建议上线后监控 API 调用量。*
